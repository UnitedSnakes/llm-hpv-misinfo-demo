# llm-hpv-misinfo
Use LLM to classify whether HPV vaccination tweets contain misinfomation.

Select parameters at

`scripts/Shanglin/Varsha/parameters.py`

Then run

`scripts/Shanglin/Varsha/run_hpv.py`

Results will be stored at

`results/Varsha`
